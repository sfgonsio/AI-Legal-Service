CORE COMPONENTS
================================================================================

1. MAIN SCRAPER: canonical_scraper.py (19 KB)
   - Production-grade Python scraper for California legal codes
   - Pulls sections from leginfo.legislature.ca.gov
   - Generates canonical JSON files for CaseCore
   - Features:
     * CLI argument handling (code abbreviation)
     * Table of contents fetching and parsing
     * Section content extraction (number, title, text)
     * Structural hierarchy parsing (division, chapter, article)
     * Automatic keyword-based tagging (10 legal domains)
     * Canonical JSON generation per specification
     * Rate limiting (1 request/second)
     * Error handling and recovery
     * Dry-run mode for validation
     * Division filtering capability
     * Resume capability (skip existing files)
     * Comprehensive logging

2. MASTER ORCHESTRATION SCRIPT: scrape_all_codes.sh (8 KB)
   - Bash script to run scraper for all 29 California codes
   - Features:
     * Sequential or parallel execution
     * Logging to scrape_all_codes.log
     * Summary reporting
     * Code validation
     * Resume support
     * Customizable base output path

3. HTML PARSING UTILITY: leginfo_parser.py (6 KB)
   - Specialized HTML parser for leginfo.legislature.ca.gov
   - Features:
     * Section number extraction with regex patterns
     * Title extraction from various heading elements
     * Full text content extraction
     * Breadcrumb parsing for hierarchy
     * Table of contents link extraction
     * Unicode support for legal symbols (§, •, →, etc.)

4. OUTPUT VALIDATION: validate_output.py (8 KB)
   - Validates all generated canonical JSON files
   - Features:
     * Format compliance checking
     * Required field validation
     * Type checking
     * URL pattern validation
     * Statistics reporting
     * Error summarization
     * Per-code reporting

5. TEST SUITE: test_scraper.py (6.6 KB)
   - Unit tests for all parsing and generation functions
   - Tests cover:
     * Section number extraction
     * Title extraction from HTML
     * Text extraction and cleanup
     * Tag generation
     * Canonical JSON structure
     * Filename generation
     * Output directory creation
     * Unicode serialization

6. CONFIG GENERATOR: generate_section_configs.py (1.8 KB)
   - Generates skeleton section configuration files
   - Creates 29 JSON files for all California codes
   - Allows pre-population with section data

================================================================================
CANONICAL JSON FORMAT
================================================================================

Each section is output as JSON with this structure:

{
  "code": "CIV",
  "section": "1550",
  "title": "Essential Elements of Contract",
  "text": "Full text of statute section...",
  "tags": ["contracts", "formation", "elements"],
  "structure": {
    "division": "3",
    "division_title": "Division 3 - Obligations",
    "chapter": "1",
    "chapter_title": "Chapter 1 - Definition",
    "article": null,
    "article_title": null
  },
  "source": {
    "type": "ca_legislature",
    "jurisdiction": "California",
    "url": "https://leginfo.legislature.ca.gov/faces/codes_displaySection.xhtml?sectionNum=1550.&lawCode=CIV",
    "editorial_note": "Auto-scraped from leginfo.legislature.ca.gov",
    "scraped_at": "2026-04-08T12:34:56.789123"
  }
}

AUTOMATIC TAGGING KEYWORDS
================================================================================

10 legal domain categories with automatic keyword matching:

1. contract       - contracts, agreements, offers, acceptance, consideration
2. liability      - liability, damages, negligence, tort, injury
3. property       - property, real property, chattel, possession, ownership
4. formation      - formation, validity, requirements, essential, elements
5. remedies       - remedy, remedies, injunction, damages, relief
6. procedure      - procedure, pleading, motion, discovery, trial, judgment
7. evidence       - evidence, witness, testimony, admissible, hearsay
8. family         - marriage, divorce, custody, child, parent, adoption
9. criminal       - crime, criminal, felony, misdemeanor, offense, sentence
10. probate       - estate, probate, will, heir, succession, testator

================================================================================
USAGE EXAMPLES
================================================================================

# Single code scrape
python canonical_scraper.py CIV

# All 29 codes
./scrape_all_codes.sh

# Dry-run (preview without writing)
python canonical_scraper.py CIV --dry-run
./scrape_all_codes.sh --dry-run

# Resume (skip existing files)
python canonical_scraper.py CIV --resume
./scrape_all_codes.sh --resume

# Specific division
python canonical_scraper.py CIV --division 3

# Parallel execution
./scrape_all_codes.sh --parallel 4

# Validate output
python validate_output.py
python validate_output.py --code CIV

# Run tests
python test_scraper.py
