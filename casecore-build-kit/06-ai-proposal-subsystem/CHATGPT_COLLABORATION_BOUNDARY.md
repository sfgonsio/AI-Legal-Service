# CHATGPT COLLABORATION BOUNDARY

## Purpose
Define the role of ChatGPT in the ongoing build process.

## Allowed
- documentation drafting
- structure refinement
- traceability assistance
- specification gap review
- naming consistency review

## Not Allowed
- uncontrolled runtime decision making
- production code authority without human review
- silent architectural overrides

## Rule
ChatGPT supports the specification and review process; it does not replace engineering ownership.
