# RESPONSIBILITY MATRIX

## Purpose
Define ownership boundaries across CASECORE build and operations.

## Primary Parties
- Product Owner
- Antigravity
- Claude subsystem
- ChatGPT support/specification assistant

## Ownership
### Product Owner
- final authority on product intent
- final authority on legal workflow fidelity
- final authority on approval/governance posture
- approval of locked-decision changes

### Antigravity
- platform architecture
- frontend and backend implementation
- workflow engine
- storage/persistence
- deployment/infrastructure
- CI/CD and integration execution

### Claude Subsystem
- bounded proposal generation
- extraction proposals
- event/tag/narrative proposals
- schema-conforming proposal envelopes
- no canonical ownership

### ChatGPT Support
- specification drafting support
- traceability support
- document refinement support
- no runtime authority

## Rule
Antigravity owns the platform. Claude participates through bounded contracts. ChatGPT supports the documentation and planning layer only.
