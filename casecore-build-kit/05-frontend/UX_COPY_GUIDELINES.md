# UX COPY GUIDELINES

## Purpose
Standardize product language.

## Preferred Terms
- supported
- unsupported
- conflicting
- traceable
- auditable
- proposal
- canonical
- reviewed
- pending approval

## Avoid
- truth engine
- constructing truth
- proven (unless explicitly appropriate and governed)
- guaranteed
- certain
- final answer

## Tone
- precise
- sober
- litigation-grade
- transparent about uncertainty
