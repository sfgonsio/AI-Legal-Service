# DECISION CHANGE CONTROL

## Purpose
Define how material specification changes are introduced.

## Requires Formal Review
- locked decisions
- schema changes
- workflow changes
- promotion rule changes
- model governance changes
- naming changes
- frontend truth-labeling changes

## Required Artifacts
- ADR or written change record
- impacted file list
- traceability update
- fixture impact note
- approval record
