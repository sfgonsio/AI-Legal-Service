# OPEN QUESTIONS

## Purpose
Capture unresolved but visible issues so they do not become silent assumptions.

## Open Questions
- exact external product name and brand
- final production hosting/deployment topology
- exact search engine implementation
- exact event bus technology choice
- exact auth provider and enterprise SSO details
- production retention periods by artifact class

## Rule
Open questions may not be silently resolved in implementation without documented change control where material.
