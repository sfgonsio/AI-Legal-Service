# EXPORT CONTRACT

## Purpose
Define governed externalization of system content.

## Allowed Export Classes
- evidence summaries
- attorney support packets
- timeline reports
- COA coverage views
- controlled data extracts

## Required Metadata
- matter id
- export timestamp
- actor
- artifact state labels where relevant
- source trace references where supported

## Rules
- exports must be auditable
- restricted content exports must be role-controlled
- proposal content must remain labeled as proposal where applicable
