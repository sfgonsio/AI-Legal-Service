# DATA LIFECYCLE POLICY

## Purpose
Define retention, archival, and deletion behavior.

## Categories
- matter metadata
- uploaded files
- canonical artifacts
- proposal artifacts
- audit records
- model interaction metadata
- exported files

## Rules
- audit records are not casually deleted
- canonical artifacts require governed archival/deletion rules
- restricted/privileged material must follow stricter handling
- deletion actions must be auditable
- legal hold behavior must override normal deletion where applicable

## MVP Rule
Default to retention and auditability over aggressive deletion.
