# EVID 210 — Relevant Evidence

**Code:** California Evidence Code  
**Section:** 210

## Text
“Relevant evidence” means evidence, including evidence relevant to the credibility of a witness or hearsay declarant, having any tendency in reason to prove or disprove any disputed fact that is of consequence to the determination of the action.

## Notes
- Foundational definition for admissibility analysis.
- Includes evidence relevant to witness credibility and hearsay declarants.
- Core intake filter for evidence, facts, and tagging.

## Platform Relevance
