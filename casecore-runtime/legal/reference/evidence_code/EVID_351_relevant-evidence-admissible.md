# EVID 351 — Relevant Evidence Admissible

## Text
Except as otherwise provided by statute, all relevant evidence is admissible.

## Notes
- Establishes default admissibility rule.
- Subject to exclusion rules (e.g., EVID 352, hearsay, privilege).

## Platform Relevance
- Default assumption for intake classification.
- Must be paired with exclusion logic.
