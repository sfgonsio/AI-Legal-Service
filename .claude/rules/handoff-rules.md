# Handoff Rules

## Product / UI path
Product Strategy Agent
-> Design System Agent
-> Case Guidance Agent
-> Code Assistant Agent
-> frontend-builder and/or backend-builder
-> QA Automation Agent
-> Delivery Agent
-> DevOps & Release Agent
-> human approver

## Legal / litigation-support path
Case Guidance Agent
-> legal-architect
-> attorney-reviewer
-> evidence-traceability-reviewer
-> deposition-strategy-reviewer

## Repo / release safety path
Delivery Agent or AgentOps Orchestrator
-> migration-manager
-> workflow-governor
-> release-gatekeeper
